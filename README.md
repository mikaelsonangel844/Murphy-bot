## 🚀 Fonctionnalités

- Réponse automatisée aux messages  
- Système de modération (antispam, anti-lien, auto-ban)  
- Téléchargement de vidéos, images et GIFs  
- Intégration de l'IA (ChatGPT)  
- Commandes personnalisées et gestion des rôles
